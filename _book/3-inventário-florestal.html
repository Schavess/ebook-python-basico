3 Inventário florestal

3.1 Levantamento Fitossociológico

3.1.1 Densidade absoluta

\[ DA_i = \frac{n_i}{A} \]

3.1.2 Densidade relativa

\[ DR_i = \frac{DA_i}{\sum_{i=1}^s (DA_i)} \]

3.1.3 Dominância absoluta

\[ DoA_i = \frac{\sum_{j=1}^{n_i} g_j}{A} = \frac{G_i}{A} \]

3.1.4 Dominância relativa

\[ DoR_i = \frac{DoA_i}{\sum_{i=1}^{S} (DoA_i)}*100=\frac{G_i}{G_T}*100 \]

3.1.5 Frequência Absoluta

\[ FA_i = \frac{U_i}{U_T}*100 \]

3.1.6 Frequência Relativa

\[ FR_i = \frac{FA_i}{\sum_{i=1}^{S}(FA_i)}*100 \]

3.1.7 Valor de Cobertura

\[ VC_i = DR_i + DoR_i \]

3.1.8 Porcentagem de Cobertura

\[ PC_i = \frac{DR_i + DoR_i}{2} \]

3.1.9 Valor de Importância

\[ VC_i = DR_i + DoR_i + FR_i \]

3.1.10 Porcentagem de Importância

\[ PC_i = \frac{DR_i + DoR_i + FR_i}{3} \]

3.1.11 Índice de Shannom

\[ H' = -\sum_{i=1}^S p_i \ln(p_i) \]

3.1.12 Índice de Morisita

\[ I_\delta = \frac{n \sum_{i=1}^S n_i (n_i - 1)}{N (N - 1)} \]
